MedFlow
Autor: Weberth Silva de Souza

---

SOBRE O PROJETO

O **MedFlow** é um sistema desenvolvido em Java com o objetivo de simular um ambiente básico de controle hospitalar.
A aplicação permite o cadastro de médicos e pacientes, bem como a associação entre eles, possibilitando a visualização dos pacientes vinculados a cada médico.

---

FUNCIONALIDADES

* Cadastro de médicos
* Cadastro de pacientes
* Associação de pacientes a médicos
* Listagem de pacientes vinculados a um médico

---

OBJETIVO

Este projeto tem como finalidade a prática e consolidação dos seguintes conceitos:

* Programação Orientada a Objetos (POO)
* Manipulação de listas com `ArrayList`
* Organização e estruturação de classes

---

COMO O SISTEMA FUNCIONA

A lógica do sistema está estruturada da seguinte forma:

* Um **Administrador (Admin)** é responsável pelo gerenciamento dos médicos
* Um **Médico** pode cadastrar e acompanhar seus pacientes
* Os dados são armazenados em memória utilizando listas, simulando o comportamento de um sistema real

---

TECNOLOGIAS E CONCEITOS UTILIZADOS

* Java
* Programação Orientada a Objetos (POO)
* Herança
* Encapsulamento
* Estruturas de dados (`ArrayList`)
