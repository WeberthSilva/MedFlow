MedFlow
Autor: Weberth Silva de Souza
------------------------------------------------------------------------------------------
Sobre o Projeto

O MedFlow é um projeto em Java criado para simular um sistema básico de controle hospitalar.
O sistema permite cadastrar médicos e pacientes, além de associar pacientes a um médico e listar essas informações.
------------------------------------------------------------------------------------------
Funcionalidades
Cadastro de médicos
Cadastro de pacientes
Associação entre médico e paciente
Listagem de pacientes vinculados a um médico
------------------------------------------------------------------------------------------
Objetivo
Praticar conceitos de programação como:

Programação Orientada a Objetos (POO)
Uso de listas (ArrayList)
Organização de classes
------------------------------------------------------------------------------------------
Como o sistema funciona
A lógica do sistema é baseada em:

Um Admin responsável por gerenciar médicos
Um Médico responsável por cadastrar e acompanhar pacientes
Armazenamento em memória utilizando listas, simulando um sistema real
------------------------------------------------------------------------------------------
Tecnologias e Conceitos Utilizados
Java
Programação Orientada a Objetos (POO)
Herança
Encapsulamento
Estruturas de dados (ArrayList)
------------------------------------------------------------------------------------------
Melhorias Futuras
